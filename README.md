# Algoritmo-e-Logica-de-programacao
Repositório que funciona como um laboratório de exercícios práticos para a disciplina de Algoritmo e Lógica de Programação do Curso de Bacharelado em Sistemas de Informação (BSI)

Objetivo: explicar que o repositório é para treinar lógica e algoritmos.
Linguagem usada: Python.
Como usar: instruções para clonar o repositório e executar os exemplos.
Sugestão de prática: cada pasta contém exemplos e exercícios para o aluno resolver.
